# E-Quail
Quail Management System

---

### Features
##### User Authentication
 - [x] Login with email and password
 - [x] Logout

##### Automated Feeding
 - [x] Detect feed level in the container
 - [ ] Auto activate feed dispenser based on feed level
 - [ ] Auto stop feed dispenser upon reaching set min
 - [x] System must display current feed level

##### Automated Water Refilling
 - [x] Continuously monitor water level
 - [ ] Auto refill water when water level drops
 - [ ] Auto stop water refilling
 - [ ] Notify users via when water level drops

##### Temperature and Humidity Monitoring
 - [x] Monitor temperature and humidity
 - [x] Realtime temperature and humidity readings

##### Light Control
 - [x] Manually turn lights on and off in UI

##### Automated Egg Counting
 - [x] The system should detect and count eggs
 - [x] Egg detection record on analytics page
 - [x] Display total eggs per day
 - [ ] Display total egg production recorded

##### Mortality Recording
 - [x] Manual encoding of mortality in the UI
 - [x] The system shall store mortality records base on datetime entry

##### Notifications and Alert
 - [ ] Auto notif when feed or water level drops
 - [ ] Auto notif for abnormal readings
 - [ ] Auto notif for anomaly in egg counting

##### Reporting and Analytics
 - [x] Display and print the current temperature and humidity
 - [x] Display and print the current feed and water level
 - [x] Display and print the current egg count for the day
 - [x] Display and print mortality reports
